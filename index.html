Introduction

Fashion image retrieval is a core capability for e-commerce platforms, powering visual search, product recommendation, and outfit generation. However, existing fashion benchmarks are largely static and product-centric, making them vulnerable to data contamination and overfitting. Even when street-look images are included, evaluation is typically framed as instance-level, single-item retrieval on a fixed test set, which is ill-suited for studying real user intents or for up-to-date assessment.

LookBench addresses these limitations as a live, holistic, and challenging benchmark for fashion image retrieval in real e-commerce settings. It is built on three key principles:

  • Continuously refreshing samples to mitigate contamination. We collect time-stamped queries from recent retail sites, enabling contamination-aware evaluation by filtering test data to post-training-cutoff crawls. We periodically refresh LookBench with newly collected queries while preserving earlier snapshots.
  • Covering diverse retrieval intents and difficulty levels. LookBench comprises four evaluation subsets spanning single-item and multi-item retrieval tasks, from clean studio product images to complex street-style outfit compositions.
  • Attribute-supervised and holistic evaluation. We construct a category-specific fashion attribute taxonomy with over 100 visually grounded properties and pre-annotate all query and ranked items. This enables region-aware evaluation and attribute-conditioned analyses of retrieval behavior (e.g., matching materials, silhouettes, and colors), supporting more fine-grained diagnosis than category-only metrics.

The current release (v20251201) comprises approximately 2,500 queries, each evaluated against a carefully curated retrieval corpus of about 60,000 images per task:

Each evaluation set is assessed using three key metrics:

  • Coarse Recall: Measures broad retrieval accuracy
  • Fine Recall: Evaluates precise matching capability
  • nDCG: Normalized Discounted Cumulative Gain for ranking quality
All metrics are evaluated at multiple recall points: @1, @5, @10, and @20 for comprehensive performance assessment.

Together with our open-sourced benchmark splits, attribute annotations, and evaluation code, LookBench enables transparent, contamination-aware comparison and reproducible research in fashion image retrieval.
